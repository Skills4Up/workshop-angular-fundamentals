# Jerarquía de Componentes en Angular

La jerarquía de componentes es un concepto fundamental en Angular que permite organizar, estructurar y reutilizar componentes para crear interfaces de usuario complejas. Comprender cómo los componentes interactúan en una estructura de árbol es esencial para desarrollar aplicaciones Angular efectivas.

## 🧩 Estructura de componentes en Angular

En Angular, los componentes se organizan en una estructura de árbol donde:

- Un componente puede ser **padre** de múltiples componentes hijos
- Un componente solo puede tener **un único padre** directo
- Los componentes **heredan contexto** de sus padres
- La comunicación entre componentes sigue patrones específicos según su relación

Comprender estas relaciones te permitirá construir aplicaciones mejor estructuradas, más mantenibles y con componentes altamente reutilizables.
