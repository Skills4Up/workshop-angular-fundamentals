# Arquitectura Modular en Angular

El sistema de módulos es un concepto fundamental en Angular que proporciona un mecanismo para organizar y estructurar las aplicaciones. Los módulos ayudan a encapsular funcionalidades relacionadas, promoviendo la modularidad, la reutilización y la carga eficiente del código.

## 🎯 Objetivos de aprendizaje

En esta sección aprenderás:

- Qué es un módulo Angular y por qué es importante
- Cómo se estructura un módulo y sus metadatos principales
- El rol del módulo raíz (AppModule) y su configuración
- Cómo declarar y organizar elementos dentro de módulos

## 💡 Conceptos clave

- **NgModule**: El decorador que define un módulo en Angular
- **Declaraciones**: Componentes, directivas y pipes que pertenecen al módulo
- **Importaciones**: Otros módulos cuyas exportaciones son necesarias
- **Exportaciones**: Elementos que pueden usar otros módulos importadores
- **Proveedores**: Servicios disponibles para la inyección de dependencias
- **Bootstrap**: El componente raíz que Angular crea e inserta en la página
